This project is about adding weapons/armors/mechanics/NPCs from fromsoftware games such as Dark Souls, Elden Ring, Bloodborne etc.
Currently featuring:
  Weapons:
    * Melee:
      - Greatsword
    * Ranged:
      - Dragonslayer Greatbow
    * Magic:
      - Soul Spear Scroll
  Items:
    * Ammo:
        - Dragonslayer Arrow
